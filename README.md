# 💰 Smart Expense Tracker

A Python-based Expense Tracker that helps users manage daily expenses using CSV file handling and data visualization.

## 🚀 Features

- Add Expense
- View Expenses
- Search Expense
- Delete Expense
- Total Expenses
- Category Summary
- Monthly Report
- Expense Statistics
- Bar Chart
- Pie Chart
- Export Report

## 🛠 Technologies Used

- Python
- CSV
- Matplotlib

## 📁 Project Structure

## ▶️ How to Run

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python expense_tracker.py
```
