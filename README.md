# 💰 Smart Expense Tracker

A console-based Expense Tracker built using Python that helps users manage daily expenses efficiently. The project uses CSV file handling for data storage and Matplotlib for visualizing expenses through charts.

---

## 🚀 Features

- ➕ Add Expense
- 📋 View Expenses
- 🔍 Search Expense
- 🗑 Delete Expense
- 💵 Calculate Total Expenses
- 📊 Category-wise Summary
- 📅 Monthly Expense Report
- 📈 Bar Chart Visualization
- 🥧 Pie Chart Visualization
- 📌 Expense Statistics (Highest & Lowest)
- 📄 Export Summary Report

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/pushpabk1010-collab/Smart-Expense-Tracker.git
```

2. Navigate to the project folder

```bash
cd Smart-Expense-Tracker
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python expense_tracker.py
```
